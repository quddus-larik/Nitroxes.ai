## Setup Project
#### **Requirements**
- NodeJS v18+
#### **React Packages**
- Tailwindcss (css writing) v3 
- React Markdown
- Axios (API handling)
- Auth0

**Clone The Git Repo**
```bash
git clone https://github.com/EnderTonol/Nitroxes.ai.git
```
**Install required packages in Both Backend & Frontend**
```bash
npm install 
```
**Run Client(Front-end) >** 
`npm start`
<br/>
**Run Server(Back-end) >**
`npm start`
